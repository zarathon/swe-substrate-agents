
## 🚀 Deployment

A landing page é automaticamente publicada pelo GitHub Pages quando há push para a branch `main`.

### Como Ativar o GitHub Pages

1. Vá em **Settings** do repositório
2. Na sidebar, clique em **Pages**
3. Em **Source**, selecione:
   - Branch: `main`
   - Folder: `/docs`
4. Clique em **Save**
5. Aguarde alguns minutos para o deploy

A URL será: `https://zarathon.github.io/swe-substrate-agents/`

## 🔧 Desenvolvimento Local

Para testar localmente:

```bash
# Opção 1: Python
cd docs
python3 -m http.server 8000

# Opção 2: Node.js
npx serve docs

# Acesse: http://localhost:8000
```

## 📝 Manutenção
